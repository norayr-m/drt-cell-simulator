# DRT Cell Simulator

> This is an amateur engineering project. We are not HPC professionals and make no competitive claims.

**Phase 4: Cellular Graph Simulator.** Cellular automata on graph topology. Each cell is an observer with local rules. Emergent global behavior from distributed local transitions. DRT as biology.

Part of the [Distributed Reconstruction Theorem](https://github.com/norayr-m) ecosystem.
